Summary

Legacy is a fairly straightforward beginner-level machine which demonstrates the potential security risks of SMB on Windows. Only one publicly available exploit is required to obtain administrator access

Initial Foothold

After conducting initial recon, identified system had open 445 port (SMB) and running on Windows XP. Searched google for CVEs applicable to Windows XP and received a list back. Utilized Metasploit to search for applicable CVE modules and found CVE-2008-4250 listed. Ran exploit and was able to get shell access as NT AUTHORITY\SYSTEM. With this level of privelege I was able to freely navigate the machine and gather the relevant flags.
